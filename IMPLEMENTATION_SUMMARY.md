## 🔐 Google Sheets Integration Setup

The contact form is ready to submit to Google Sheets. Follow these steps:

### Quick Setup (5 minutes):

1. **Create Google Sheet**
   - Go to sheets.google.com
   - Create new sheet with columns: Full Name, Mobile Number, Email ID, Town, State, Country, Looking For, Description, Timestamp

2. **Create Google Apps Script**
   - Tools → Script Editor
   - Paste the provided script (see SETUP_GUIDE.md)
   - Save & Deploy as Web App

3. **Update Contact Form**
   - Copy the Web App URL
   - Edit `src/pages/Contact.jsx`, line ~43
   - Replace `YOUR_SCRIPT_ID` with your Script ID

4. **Done!** Form now submits to your Google Sheet

📖 **See SETUP_GUIDE.md for detailed instructions**

All pages are easy to customize:

- Edit page components in `src/pages/`

### Modify Colors:
- Edit `tailwind.config.js` to change gradients
- Update Tailwind color classes in components

### Add More Pages:
- Create new file in `src/pages/`
- Add route in `src/App.jsx`
- Add to navbar navigation

### Change Animations:
- Adjust delay values in Framer Motion `transition` props
- Modify keyframes in `tailwind.config.js`
